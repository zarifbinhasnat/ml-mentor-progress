# ML Mentor Progress

Tracks daily progress for the ML Mentor bot.

> **Note:** This repository is a standalone progress tracker and is **not related** to [class-ai-companion](https://github.com/zarifbinhasnat/class-ai-companion).

## Structure

- `progress.json` — persists the current day, last concept covered, algorithm language preference, and last run date for the ML Mentor bot.
